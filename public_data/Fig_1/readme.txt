We provide data needed of scalaried neutron stars in massless DEF theory with AP4 EOS.
The coupling parameter beta is showed in the file name.

CE: central mass density
RE: stellar radius in Einstein frame
RJ: stellar radius in Jordan frame
MA: ADM mass
MB: baryonic mass
PC: scalar field value at stellar centar
PS: scalar field value at stellar surface
AL: charge-to-mass ratio
MI: moment of inertia
